# 📊 PR.3 – Spread Locator: Statistical Distribution Analysis

## 📖 Project Overview

This project performs **statistical analysis and probability distribution modeling** on a **Spread Locator transaction dataset**.

The notebook demonstrates how different statistical distributions can be applied to real-world transaction data and includes hypothesis testing, probability calculations, normality checking, and visualization.

---

## 🎯 Objectives

- Analyze transaction dataset
- Perform exploratory data analysis (EDA)
- Apply probability distributions
- Visualize statistical behavior
- Detect outliers
- Test data normality
- Calculate probabilities using statistical models

# 📈 Statistical Tasks Performed

## ✅ Task 1 — Bernoulli Distribution

- Convert transaction status into Success/Failure
- Calculate probability of success
- Plot Bernoulli PMF

---

## ✅ Task 2 — Poisson Distribution

- Daily transaction count
- Mean transaction frequency
- Poisson Probability Mass Function
- Visualization

---

## ✅ Task 3 — Log-Normal Distribution

- Fit Log-Normal model
- Estimate parameters
- Probability Density Function
- Distribution plot

---

## ✅ Task 4 — Q-Q Plot & Shapiro Test

- Q-Q Plot
- Shapiro-Wilk Normality Test
- Statistical interpretation

---

## ✅ Task 5 — Box-Cox Transformation

- Normalize skewed data
- Lambda estimation
- Before & After comparison

---

## ✅ Task 6 — Z-Score Analysis

- Compute Z-Score
- Detect unusual observations
- Probability calculation

---

## ✅ Task 7 — Probability Density Function (PDF)

- Estimate Normal Distribution
- Plot PDF
- Mean & Standard Deviation

## ✅ Task 8 — Cumulative Distribution Function (CDF)
